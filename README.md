# 🔍 Deep Research Agent

An AI-powered competitor analysis agent built with Claude AI and DuckDuckGo search.

## What it does
- Takes any topic as input
- Autonomously searches the web multiple times using DuckDuckGo
- Uses Claude AI to synthesise findings into a structured report
- Covers: Positioning · Differentiation · Impact · Market Sentiment
- Generates a downloadable HTML report

## Built with
- [Anthropic Claude](https://anthropic.com) — AI brain
- [DuckDuckGo Search](https://pypi.org/project/duckduckgo-search/) — web search tool
- [Streamlit](https://streamlit.io) — web interface

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
